# CodeBoy版开源中国
  我个人向来比较喜欢逛开源中国，因为不时会发现一些好的新闻或者源码，也能学到不少东西。或者开源中国的Android客户端打开多了，发觉界面还是有点忧伤，而且已经是属于很旧的界面风格。因此萌发了重写开源中国的客户界面的念头，另一方面，也可以给部门的成员学习一下。当然万事起头难，最让我感到头痛的是，开源中国的Android客户端源代码关于界面方面显然是扎堆的起来的，已经很难去维护或新增功能，不过庆幸逻辑层与Api封装得比较好，不用怎么改就可以直接套用，也涨姿势了。

  而CodeBoy版的开源码中国，基本上不再沿用旧版的界面设计方式（除了Adapter之外），而采用了全新的而且最流行的设计方式，Fragment+ViewPage+ActionBar(android-support-v7兼用包)。
  项目启动是从2014年4月19日开始,那是一个安静的周末，我悠闲地喝上一杯茶，然后没日没夜地编码。。哈哈，其实，每天利用业余时间去一点点地编写，积累下来的效果还是不错的。希望大家可以完善这个项目，也期待开源中国越来越好。
